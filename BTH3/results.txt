===== KẾT QUẢ KIỂM THỬ - BIÊN & KHÔNG HỢP LỆ =====

1. Chu vi hình chữ nhật
Input (1,1) -> 4
Input (0,5) -> Dữ liệu không hợp lệ
Input (-2,3) -> Dữ liệu không hợp lệ

2. Diện tích hình chữ nhật
Input (1,1) -> 1
Input (0,2) -> Dữ liệu không hợp lệ

3. Phương trình bậc 2
Input (1,0,1) -> Vô nghiệm
Input (0,2,1) -> Không phải PT bậc 2

4. Số ngày trong tháng
Input (13,2024) -> Tháng không hợp lệ

5. Số nguyên tố
Input 1 -> False
Input -5 -> False

6. Tổng S = 1 - 2 + ... + n
Input 1 -> 1
Input 0 -> Không hợp lệ

7. UCLN
Input (0,5) -> 5

8. Tổng giai thừa
Input -1 -> Không hợp lệ

===== KẾT LUẬN =====
- Các trường hợp dữ liệu không hợp lệ được xử lý đúng
- Chương trình không bị lỗi hoặc crash
- Các giá trị biên hoạt động chính xác
